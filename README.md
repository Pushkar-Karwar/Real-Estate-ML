# Real Estate Price Prediction using Machine Learning

This project predicts house prices using multiple Machine Learning models.

## Models Used
- Linear Regression
- K Nearest Neighbors (KNN)
- Random Forest Regressor

## Workflow
1. Data preprocessing
2. Feature engineering
3. Model training
4. Model comparison
5. Best model selection

## Evaluation Metric
R² Score

## Dataset

The dataset contains **4140 rows and 18 columns**, including various features related to residential properties such as bedrooms, bathrooms, square footage, location, and other house characteristics used to predict real estate prices.
